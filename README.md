
# 일기장 서비스 프로젝트
---
자바 스프링 부트로 일기장 서비스를 구현했다.




# 개발 환경
---
* Intellij IDEA Ultimate 2022.2.1
* Java 17
* Maven 4.0.0
* Spring Boot 2.7.4




## 구현 내용
---
1. JWT 구현 (토큰)
2. 쿠키 구현
3. MVC 모델로 구현
4. ManytoOne, OnetoMany로 ENTITY 설정
5. CRUD를 활용하여 로그인 기능과 Diary 저장 기능 실행
6. DAO, DTO, VO, ENTITY
7. 예외처리 CustomExceptionHandler 구현
8. Cloudtype과 연결하여 외부 cloud로 연결
